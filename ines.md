# Generador de Excusas de Trabajo

Aplicacion web simple para generar excusas laborales de forma aleatoria.

Cada resultado incluye:
- Excusa lista para usar.
- Situacion en la que encaja.
- Motivo de por que puede funcionar.

## Como usarlo

1. Abre el archivo `index.html` en tu navegador.
2. Pulsa **Generar excusa** para obtener una nueva opcion.
3. Pulsa **Copiar texto** para copiar la excusa completa.

## Archivos

- `index.html`: estructura de la interfaz.
- `styles.css`: estilos y animaciones.
- `script.js`: lista de excusas y logica del generador.
